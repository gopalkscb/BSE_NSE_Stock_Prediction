# Product Overview

BSE/NSE Stock Prediction is a Python-based application that predicts stock ticker prices and trends for Indian stock markets — both the Bombay Stock Exchange (BSE) and the National Stock Exchange (NSE).

## Core Purpose
- Fetch and process historical stock data for BSE/NSE listed tickers
- Train and evaluate machine learning or deep learning models for price prediction
- Expose predictions through a UI or API for end users

## Target Users
Traders, analysts, and data scientists interested in Indian equity market forecasting.
